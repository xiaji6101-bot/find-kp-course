先搞清楚：什么是KP？

▾
KP = Key Person，指最终拍板决定合作的那个人，不是职位最高的，不是最热情的，而是说"行"就行、说"不行"就不行的人。
酒店类型KP通常是
单体低星酒店 / 民宿老板本人（或老板娘）
单体高星酒店总经理 / 店长
集团连锁酒店收益总监 / 销售总监
⚠
假KP识别：对方很热情、说"没问题我来安排"，但没有决策权。
识别方法：直接问——"关于和我们合作这件事，谁来负责最终决定？"

第一部分：上门拜访

核心逻辑

直接走进酒店，第一个遇到的人（前台/保安/收银员）往往不是KP，但一定要把眼前这个人当成你此刻最重要的人，才有可能找到KP。

关键成功要素

入门第一步是与前台、保安等一线人员建立好感。建立好感的方式包括寒暄、赞美、递奶茶、聊天等互动。

操作流程

1
建立好感
2
寻求帮助
3
请求引荐
场景：进门后如何与一线员工互动并找到KP

话术：建立好感 → 寻求帮助 → 请求引荐

第一步：简单寒暄，破冰建立好感
"今天入住的人挺多啊，你们忙坏了吧？"
"你们前台布置得挺温馨的，生意不错吧？"
"×姐，看你这么精神，今天忙到现在还没休息吧？"

（聊几句后自然过渡）
"对了，我是飞猪的小李，想拜访一下你们负责人，他在吗？"
（如果对方说不在/不确定）
"那他一般什么时候比较方便？我改天再来，或者能留个联系方式吗？"

第二步：聊几句后请求引荐KP
"×姐/×哥，我有件事想请你帮个忙，对我来说挺重要的。能不能帮我引荐一下你们老板/总经理？我们有个对你们酒店有帮助的合作方向，想当面聊聊。"

如果对方犹豫：
"你觉得引荐这边会有什么顾虑吗？你说说，我们一起想想怎么解决。"

得到同意后立刻约时间：
"太好了！你看是现在方便，还是我约个具体时间过来比较好？"

第二部分：多次拜访

多次拜访策略（一定要坚持）

第一次
初步接触，留下印象。
和前台/保安简单寒暄，介绍自己，让对方记住你的名字和长相。
第二次
记住名字和长相，见到主动打招呼。
可携带小礼物（如奶茶、烟），用熟悉感降低防御。
第三次
进一步加深关系，直到拿到KP信息。
让一线员工从"不认识的BD"变成"熟悉的人"，主动帮你创造接触KP的机会。
💡
小Tips：眼前的一线员工，就是你当下唯一的KP。只有他真心愿意帮你，才能帮你找到真正的KP。
把每一个遇到的人都当成重要的人——这不只是技巧，这是真正的尊重。

核心逻辑：价值主张法

不是"隐瞒身份"，而是先以行业交流者/资源提供者身份切入，提供真实有用的价值，让对方主动想了解你是谁，再自然亮明BD身份。

3

话术示例

对KP：
"于总您好，我发现最近商圈流量下滑明显，我们有个高德引流活动，预计能提升出租率15%，您有兴趣了解一下吗？"

对前台（借力传话）：
"美女，我看你们中午忙得没时间吃饭，我们有套AI系统能帮你分担接待压力，每天能省出一小时休息时间，要不要我帮你介绍下负责人？"

📊 行业数据

让对方觉得你有信息

找KP是非观

▾
🚫
以下行为不只是"效果不好"，部分涉及风控红线和合规问题，一旦出现可能影响账号、影响数据，甚至上升至法务层面。
🚫 风控 / 合规红线
  • 用虚假身份或假公司名义签约
  • 承诺平台无法兑现的数据/排名/返佣
  • 私下截流平台订单或截佣
  • 帮商家操作虚假评论、刷单
  • 透露其他商家的商业数据或价格信息
  • 未经授权使用平台名义做承诺
🚫 坑蒙拐骗类（建立在谎言上的关系走不远）
  • 谎称自己是"旅行社/渠道商/采购商"去套信息
  • 假装是住客去探情报，然后突然"翻牌"说自己是BD
  • 夸大平台流量、效果、GMV数据
  • 对商家说"签了马上就有单"（无法保证的事不能承诺）
  • 用其他商家的隐私信息作为谈判筹码
  • 答应给商家"特殊政策"但实际上没有
⚠ 认知陷阱（容易被坑）
  • 把非KP当KP一直跟进——跟前台聊了3次，老板从没出现
  • 把"热情"当"有决策权"——对方很积极，但他做不了主
  • 没确认身份就开始讲方案——讲了半小时，对方是实习生
  • 认为"见到KP=谈成了"——见面只是开始，不是终点
  • 以为一次拜访就能成单——平均需要3-5次接触才能建立信任
⚠ 低效行为（浪费时间）
  • 没做任何准备就冷拜访，不知道对方是谁
  • 谈完一家，走时没有要引荐（最高效的路没走）
  • 对方说"我考虑考虑"就等着，没有约下一步
  • 只发消息不打电话，只打电话不约见面
  • 在不合适的时间去谈（午休、旺季入住高峰）
  • 见到人就马上推产品，没有先建立信任

课堂外知识分享 · "找到决策者"

▾
核心观点：每个人都有一个核心需求

通常，我们认为自己最关心的是钱、效率、结果。但其实，在所有人际互动中，有一个更底层的需求一直在驱动人的行为：

被尊重、被理解、被当成重要的人

当你走进一家酒店，前台每天接待无数"来推销的人"，他们的默认状态是防御和敷衍。但如果你能让他感受到——你不是来利用他的，你是真的把他当回事——他就会放下防御，甚至主动帮你。

在实际拜访中，这意味着什么？

不要：进门直接问"你们老板在吗？"——这让对方感觉自己只是一个"中转站"

要：先打招呼、称赞一件具体的小事（"你们大堂布置得挺有品味的"），然后自然引入目的——对方会感觉自己是被重视的人，而不是被绕过的障碍。

关于"找到真正的决策者"

找KP的难点不只是"找到谁"，更是"怎么让那个人愿意见你、愿意和你说话"。

越是高层的人，越不缺资源，他们最缺的是：真正理解他处境的人，和真正有价值的信息。

所以在每次拜访前，问自己一个问题：

👆 一个问题

如果我是这家酒店的老板，今天这个BD来见我，我有什么理由想见他？他能给我带来什么？

如果你回答不上来，那就是你还没准备好去拜访。

好的关系从"给予"开始

每一次拜访，先想：我能给这个人什么有用的东西？（信息/数据/行业视角/切实帮助）

当对方感受到你是来"给"的，不是来"取"的，信任就会建立得快很多。

这也是第4条路（先提供价值再亮身份）的底层逻辑。
